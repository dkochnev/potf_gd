About

When two armies share one castle, only one will survive. Capture the opponent’s commander and
own the castle.

What is it?

“Castle Fight” is an abstract strategy game. That means it is similar to checkers or chess.

For players from 12 years of age. Children under 3 years should not be allowed as game contains small parts.

Playtime is about 6-30 min. The game can lasts longer or shorter, but in most cases it takes the mentioned time.

How to play it?

The game comes in a box that contains the board, the units and the rulebook.

In order to play the game you will need the board and the units. The rules are written in the rulebook. It contains a thorough description of actions. There is a description how to set up the game, what the victory conditions are, how to make moves and fight.
